# 📞 サポートポリシー (Support Policy)

Luminos-Studio-Dev のプロジェクトをご利用いただきありがとうございます。質問、問題、提案がある場合は、以下のガイドラインに従って適切な窓口をご利用ください。

## 📝 1. サポート窓口

| 目的 | 適切な窓口 | 補足事項 |
| :--- | :--- | :--- |
| **技術的な質問** (使い方、設定など) | **Discord サーバー**の `#help` チャンネル | リアルタイムでコミュニティから回答が得られやすいです。 |
| **一般的な提案やフィードバック** | **Discord サーバー**の `#feedback` チャンネル | 開発チームが定期的に確認しています。 |
| **バグ報告** | **GitHub Issues**（バグ報告テンプレート） | 再現手順を添えてください。[CONTRIBUTING.md] を参照。 |
| **新機能の要望** | **GitHub Issues**（機能要望テンプレート） | 提案の背景にある問題を明確にしてください。[CONTRIBUTING.md] を参照。 |
| **セキュリティ脆弱性の報告** | **GitHub Private Reporting** または `security@luminos-studio-dev.org` | 絶対に公開の場での報告は避けてください。[SECURITY.md] を参照。 |

## ⚙️ 2. サポート対象範囲

現在、Luminos-Studio-Dev は**最新の安定バージョン**（通常、メインブランチまたは最新のタグ付けされたリリース）のみをサポートしています。

* **サポート対象外の項目:**
    * カスタムコードや独自の環境で発生した問題。
    * 古すぎるバージョンで発生している既知のバグ。
    * 私たちのプロジェクト以外のサードパーティ製ツールの問題。

まず、[FAQ] や既存の Issue を検索し、自力での解決を試みることを推奨します。
