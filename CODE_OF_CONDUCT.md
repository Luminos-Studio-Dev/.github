# 🤝 行動規範 (Code of Conduct)

Luminos-Studio-Dev のすべての参加者（メンバー、貢献者、ユーザー）は、年齢、身体的特徴、人種、宗教、性別に関わらず、**ハラスメントのない安全で歓迎的な環境**を提供することに努めます。私たちは、建設的な議論と協力的な精神を重視します。

## 期待される行動

* **敬意と配慮:** 他者に対し、敬意をもって接してください。建設的な批判や意見の違いは尊重しますが、個人攻撃は許容されません。
* **協力的な態度:** コラボレーションを重視し、他のメンバーの質問や問題解決に積極的に協力してください。
* **誠実なコミュニケーション:** 誤解を避けるため、コミュニケーションは明確かつプロフェッショナルに行ってください。
* **受け入れ:** 私たちのコミュニティは、多様な背景を持つ人々を歓迎します。

## 許容されない行動（ハラスメントの例）

以下の行動は、コミュニティ内で許容されません。

* 性的な言葉や画像の使用。
* 個人に対する攻撃的なコメント、嘲笑、侮辱。
* いかなる形態の差別（人種、宗教、性的指向、障害などに基づくもの）。
* 執拗な威嚇行為、ストーカー行為。
* 公共の場での継続的な議論の中断。

## 報告と対応

行動規範に違反する行為、またはハラスメントと感じた場合は、速やかにコアメンバーにプライベートな方法で報告してください。

* **報告窓口:** `security@luminos-studio-dev.org` または Discord のオーナーへのダイレクトメッセージ
* 報告を受けた後、コアチームは状況を調査し、違反者に対して一時的または永久的な追放を含む適切な措置を講じます。
